# Aurum

Aurum is a lightweight, modular reasoning system built for learning AI fundamentals on low-end hardware (including Android via Pydroid 3).

## Features
- Smalltalk and basic conversation
- Safe math evaluation
- Simple long-term memory with confidence
- Action comparison using a simulated world model
- Rule-based reasoning
- Rule-based reasoning

## Status
- Current version: v0.3.1
- This project is under active development

## Philosophy
Aurum is an educational experiment. Heavy models and large datasets are intentionally avoided. General knowledge will be added after v1.0.0.

## Requirements
- Python 3.9+
- No external dependencies
